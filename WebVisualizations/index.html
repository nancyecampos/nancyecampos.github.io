Summary Latitude vs. X

Welcome to the latitude analysis dashboard! The purpose of this site is to provide an anlysis on how weather changes as you get closer to the equator. The data for this dashboard is pulled from OpenWeatherMap API, which was used to assesmble a dataset on over 500 cities.

Matplotlib was used to plot various aspects of weather vs. latitude. Variables assessed included temperature, cloudiness, wind speed, and humity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of trends and correlations.
